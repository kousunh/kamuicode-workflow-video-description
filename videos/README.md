# Videos Directory

このディレクトリに処理したい動画ファイルを配置してください。

## サポートされているフォーマット

- MP4 (推奨)
- AVI
- MOV
- MKV
- WebM

## サンプル動画の配置

```bash
# 例：サンプル動画をこのディレクトリにコピー
cp /path/to/your/video.mp4 ./sample.mp4
```

## 注意事項

- 大きなファイル（50MB以上）は処理に時間がかかる場合があります
- GitHub Actionsの実行時間制限（6時間）に注意してください
- 動画の解像度は1920x1080が推奨です（他の解像度でも動作します）
